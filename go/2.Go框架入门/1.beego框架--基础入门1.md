### 前言



### 1.安装Go环境


```
# 安装go
brew install go

# 升级go
brew upgrade go
```

### 2.安装或者升级Beego和Bee的开发工具

```
# 安装beego
go get github.com/astaxie/beego

# 安装bee工具
go get github.com/beego/bee
```
